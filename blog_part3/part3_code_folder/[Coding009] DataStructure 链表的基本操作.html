重点操作：找到前一个元素！

 

重点强调

这里我整理学习的是C语言版的单链表实现。当然还有复杂的链表，比如双向链表。后面不一定会整理，但是这个单链表一定是重中之重！

  • 重中之重的重中之重：在编程的过程要有意识地注意找到待操作元素的前一个元素的位置。

  • 后面会遇到复杂的代码操作，可能会有很多指针，p, q, r, s...等等。要注意辨析每个指针的作用！
